Resume
Filtero

Filtero is a simple and user-friendly web app for applying preset filters to images. It was my final project for the CS50 course. The backend is built with Django and Django REST Framework, using JWT for authentication, while the frontend is made with React. The backend is hosted on Render and the frontend on Vercel, so it might take a short moment to load when you first open it.

Visit page GitHub Repository
Robot Controller

This was one of my first projects. It’s a Django app that lets you control a robot remotely by sending commands over TCP sockets. The robot runs on a Raspberry Pi acting as a TCP server, which listens for commands and controls movement using the PiCar-X library. A fun way I got into combining software with hardware.

GitHub Repository
Lehrstellen Bewerber

A small tool that automates writing and sending apprenticeship (or general job) applications. It saves time by automatically filling in company details, dates, job titles, and even optional motivation text. It connects to Gmail, builds emails using MIME, and edits Word documents with python-docx, all through a simple CLI interface. I used it myself and shared it with classmates.
